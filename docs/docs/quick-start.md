# Getting Started

You can create a new oBerry project with:

::: code-group

```sh [pnpm]
pnpm create oberry
```

```sh [npm]
npm create oberry
```

```sh [yarn]
yarn create oberry
```

```sh [bun]
bunx create-oberry
```

:::

Or add oBerry to an existing one:

::: code-group

```sh [pnpm]
pnpm add oberry
```

```sh [npm]
npm install oberry
```

```sh [yarn]
yarn add oberry
```

```sh [bun]
bunx add oberry
```

:::
